# What is Generative Media?

Generative media refers to content created entirely by AI, from scratch, without relying on pre-made images, recordings, or stock media. When you provide a prompt such as “a surreal forest scene” or “a mellow, ambient melody”, the AI doesn’t pull elements from existing files. Instead, it uses patterns and knowledge it has learned to create something entirely new and unique. # How are credits counted?

Credits are used when creating videos that include Generative Media and are calculated based on the duration of the video and the type of generative media selected.

When you enter your prompt and click 'Generate a video', you’ll see the number of credits that will be deducted from your usage, as shown below.

# Credit Cost Breakdown

Here's a breakdown of how credits are utilized for the different types of generative media:

# Videos created using Generative Lite:

- **Cost**: 8 credits per minute

|                     |                 |
| ------------------- | --------------- |
| Video Duration      | Credits Counted |
| 30 seconds          | 4 credits       |
| 1 minute            | 8 credits       |
| 1 minute 30 seconds | 12 credits      |

# Videos created using Generative:

- **Cost**: 20 credits per minute

|                     |                 |
| ------------------- | --------------- |
| Video Duration      | Credits Counted |
| 30 seconds          | 10 credits      |
| 1 minute            | 20 credits      |
| 1 minute 30 seconds | 30 credits      |

# Videos created using Generative Ultra:

- **Cost**: 40 credits per minute

|                     |                 |
| ------------------- | --------------- |
| Video Duration      | Credits Counted |
| 30 seconds          | 20 credits      |
| 1 minute            | 40 credits      |
| 1 minute 30 seconds | 60 credits      |

# With actors (Human or Pro actors):

- **Cost**: Additional 5 credits every 30 seconds

|                     |                 |
| ------------------- | --------------- |
| Video Duration      | Credits Counted |
| 30 seconds or less  | + 5 credits     |
| 1 minute            | + 10 credits    |
| 1 minute 30 seconds | + 15 credits    |

# Good to know:

If you edit a video using our **remix** feature, credits will be utilized for the new or modified media: 1 image = 0.2 credits, a 5-second clip = 1.67 credits, and a 3-second clip = 1 credit.
