---

The "My Portfolio" view is a user-dynamic filter which makes it easy for each Planhat user to filter their portfolio based on some team member(s) field. This will help you if you otherwise would like a lot of private filters!

[Watch video](https://www.loom.com/embed/11bf1a64cdbc4fc08d5ee1d5032abe85)

Some key things to know:

- You can add multiple "Portfolios" if you look at it from multiple perspectives, eg "Account Owner / Co-Owner" vs. "Implementation Lead"

- It is editable under the Company object in the Data Module (but can then be applied wherever there is a Search bar, see video above)

- This can be combined with other filters, and used across all objects

- 👑 Pro tip: combine with [filter lock](https://support.planhat.com/en/articles/5591264-faq-how-to-use-the-filter-lock-feature), and suddenly all of Planhat is just for you!

- The definition of "My portfolio" can be customised by admins (for everyone) and for each user, based on e.g., any custom fields you may have to indicate customer relationships (e.g., "Implementation Manager" or "Solution Architect")

- You can use it wherever there is a Search Bar, but it's not usable elsewhere

**Where is my Portfolio filter?&#x20;**&#x54;here is no permission on whether the Portfolio filter, so it means that an admin has removed it. Go to Company model/object in the Data Module, hover over "General", press "Add Portfolio filter" - voilá!
