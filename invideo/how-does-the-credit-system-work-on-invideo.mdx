title: How does the credit system work on invideo?

---

We’ve simplified how usage is measured on invideo.

Credits are invideo's internal currency, and they're used each time you create or edit a video. The number of credits used depends on how long your video is and what kind of content you include, like AI actors, express clones, or just stock footage.

Credits are charged at the time of video creation, based on the duration of your video (calculated per second).

# 💡 Note:

You can create videos either with an actor from our library or with your own cloned AI avatar, or even without one. Additionally, by default, each ad is limited to a minimum duration of 30 seconds.

---

Here's a breakdown of how the credits are utilised:

## Creating ads or videos using "Generative" media (AI-generated videos)

## Videos created using **Generative**:

- **Cost**: 20 credits per minute

|                     |                     |
| ------------------- | ------------------- |
| **Video Duration**  | **Credits Counted** |
| 30 seconds          | 10 credits          |
| 1 minute            | 20 credits          |
| 1 minute 30 seconds | 30 credits          |

# 🤓 Good to know:

Suppose you edit a video created using our **[remix](https://help.invideo.io/en/articles/10429349-how-can-ai-generated-visuals-be-modified)** feature, whether by modifying the generated media or adding more chapters from the Script section or changing the media duration. Here's how credits will be utilised: 1 image = 0.2 credits, a 5-second clip = 1.67 credits, and a 3-second clip = 1 credit.

## Videos created using **Generative Lite**:

- **Cost**: 8 credits per minute

|                     |                     |
| ------------------- | ------------------- |
| **Video Duration**  | **Credits Counted** |
| 30 seconds          | 4 credits           |
| 1 minute            | 8 credits           |
| 1 minute 30 seconds | 12 credits          |

## Videos created using **Generative Ultra**:

- **Cost**: 40 credits per minute

|                     |                     |
| ------------------- | ------------------- |
| **Video Duration**  | **Credits Counted** |
| 30 seconds          | 20 credits          |
| 1 minute            | 40 credits          |
| 1 minute 30 seconds | 60 credits          |

## 🎭 With actors **(Human or Pro actors)**:

- **Cost**: Additional 5 credits every 30 seconds

|                     |                     |
| ------------------- | ------------------- |
| **Video Duration**  | **Credits Counted** |
| 30 seconds or less  | + 5 credits         |
| 1 minute            | + 10 credits        |
| 1 minute 30 seconds | + 15 credits        |

**To get started with avatars and actors:**

You can find detailed questions and answers on how to create your avatar **[here](https://help.invideo.io/en/collections/12926671-avatars)**, and if you're looking to create videos using actors from our library, this **[guide](https://help.invideo.io/en/articles/11537211-how-can-i-create-a-video-using-an-actor-from-the-invideo-library)** will walk you through the steps.

---

## **Creating videos using "Stock" media ads or videos using a Human or Pro actor:**

- **Cost:** 5 credits per 30 seconds + video mins are calculated.

|                    |                   |
| ------------------ | ----------------- |
| **Video Duration** | **Actor credits** |
| 30 seconds or less | + 5 credits       |
| 1 minute           | + 10 credits      |
| 5 minutes          | + 50 credits      |
